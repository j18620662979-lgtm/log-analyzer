# log-analyzer

**面对IOT设备测试的日志分析网站**

**[👉 点此在线体验](https://j18620662979-lgtm.github.io/log-analyzer/)**

**主要功能**

1.上传 / 粘贴日志：支持拖拽文件、选择文件或直接粘贴，文件仅在浏览器本地处理

2.选择分析模式：通用日志分析 / 稳定性分析 / 网络分析 / OTA分析（仅作为统计规则选择）

3.统计概览：日志总量、INFO / WARNING / ERROR 数量、各模式专属指标

4.异常分类：按日志内容简单分类（崩溃 / ANR / OOM / 异常 / 重启 / 网络 / 告警 / 错误）并计数展示

5.关键异常日志：列出异常所在行号、时间、类型与原文片段

6.筛选过滤：按异常类型 / 日志级别 / 关键字快速定位所需信息

7.报告导出：一键复制或下载 Markdown 统计报告

**技术栈**

HTML + CSS + JavaScript

零外部依赖：单文件即可离线打开运行，无需安装与构建

安全本地化：日志不上传任何服务器，全程浏览器内处理

**适用场景**

稳定性测试、网络兼容测试、OTA 升级测试等专项测试后的日志统计与问题复盘辅助
